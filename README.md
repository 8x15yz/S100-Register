# S100-Registery system

## 개요
- s-100의 registry 클래스 다이어그램을 파이썬으로 구현하였습니다.
- 개발환경 : python 3.9



## 디렉토리 구성

```
📁 S100-Register
	ㄴ1️⃣ register_model.py
	ㄴ2️⃣ run.py
	ㄴ3️⃣ main.py
	ㄴ4️⃣ sample_data.py
```

- `register_model.py`에 클래스 다이어그램을 구현하였습니다.

- `run.py` 에서 데이터를 입력하고 조회하는 기능을 간단하게 만들어놓았습니다.



## 실행 방법

`\S100-Register` 경로에서 다음의 코드를 실행시킵니다. 

`python main.py`
